# Am Israel Rides - Jewish Motorcycle Community

## About
A Jewish motorcycle community based in Boca Raton, Florida. We organize monthly rides and build connections within the Jewish riding community.

## First Group Ride
**Sunday, June 7th, 2025 at 9:00 AM**

## Website Setup

### Option 1: Static Hosting (Recommended - Free)
1. Upload these files to GitHub Pages, Netlify, or Cloudflare Pages
2. Connect your domain `amisraelrides.com`

### Option 2: Traditional Web Hosting
1. Upload files via FTP to your hosting provider
2. Point `amisraelrides.com` to your host

## Files
- `index.html` - Main website
- `style.css` - Styling
- `images/` - Directory for photos (upload group ride photos here)
